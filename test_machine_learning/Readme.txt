sudo apt-get install libfreetype6-dev libjpeg8-dev libpng-dev
pip install matplotlib


test_machine_learning is a simple command line tool which we have written and used to test some parameters of the machine learning algorithm it has 3 functionality
-p show the plot of the classification helpfull only fo 2D features
-c show confusion matrix and othe info
-t use a grid search algorithm to find best gamma and C parameter on an SVC model
